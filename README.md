# Akash Digital PrintHub V4

Run:

```cmd
py -m pip install -r requirements.txt
py app.py
```

Open: http://127.0.0.1:5000
Admin: http://127.0.0.1:5000/admin
Rates: http://127.0.0.1:5000/settings

V4 Features: multiple file upload, PDF page count, rate settings, admin dashboard, order detail, printer suggestion.
